1. 优惠收集分享系统

系统设计

        为了提高优惠收集的效率，同时满足用户查找优惠的便捷性。在优惠收集上，存在多种方式，包括手动录入、爬虫抓取和接口调用。存在多种类别的优惠信息，包括外卖、出行、银行等优惠信息。最后将收集的优惠信息分享给用户，用户便能便捷的查找各优惠信息。优惠收集和分享系统的整体设计如下图。

        系统的设计灵感来源于一种“聚合”的思想，即“将多种输入聚合成一个输出”。比如QQ浏览器的全网搜索，聚合了百度搜索和搜狗搜索；高德地图的打车模块，聚合了曹操出行、如祺出行和T3出行等许多打车平台；Apifox聚合了Postman、Swagger、Mock和JMeter技术等。

该系统使用的技术栈如下：

  • ZooKeeperh和Dubbo
  • Maven项目
  • Spring Boot
  • Spring Security
  • Apache POI
  • Quartz
  • MyBatis
  • Redis
  • 前端Vue + NutUI + ElementUI + Echarts
  • 第三方服务与AI
